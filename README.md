## О чём пет-проект Plexus

Веб-приложение - **мультитенантная база знаний с модулем проектов и финансов**. По сути, попытка объединить лучшее из Obsidian, заметки, локальное хранение, гибкое оформление, с управлением проектами и личными финансами в одном продукте. Изначально называлось Lifeloga, переименовано в Plexus после анкетирования - 45% назвали старое имя неудачным.

**Три ключевых модуля:**
- база знаний - заметки со структурой через директории, категории, теги. Мультитенантность, импорт из других баз знаний, локальное хранение/офлайн, гибкое оформление. Включён в MVP;
- модуль проектов - конечные последовательности решений для достижения целей, ремонт, выполнение сложного заказа и т.п., возможна привязка денежных средств. Включён в MVP;
- модуль финансов - учёт затрат и доходов на счетах. Отложен на следующие релизы, обязателен импорт транзакций из банков на момент релиза.
